O que é o SUS?

O Sistema Único de Saúde (SUS) é o sistema público de saúde do Brasil, que garante o acesso gratuito a todos os cidadãos a uma variedade de serviços de saúde, incluindo atendimentos de urgência e emergência, consultas médicas, exames e internações.

Este sistema é fundamental para a promoção da saúde, prevenção de doenças e cuidados médicos contínuos, assegurando a todos os brasileiros o direito ao acesso à saúde.


Quando Procurar Cada Tipo de Serviço de Saúde?

  • Unidade de Pronto Atendimento (UPA)

    A UPA é destinada a casos de urgência, como dores fortes, febres altas, cortes profundos, dificuldades respiratórias, etc. Nela, são feitos atendimentos rápidos para estabilizar o paciente e, se necessário, encaminhá-lo para outros serviços.
